QUICK START COMMANDS
================================================================================

# 1. Get API key from: https://api.anthropic.com

# 2. Create directory
mkdir investment-portfolio-tracker
cd investment-portfolio-tracker

# 3. Copy these files into that directory:
#    - AntiGravity_Prompt.md
#    - generate-api.js
#    - (create package.json - see QUICK_START.md)

# 4. Install Anthropic SDK
npm install @anthropic-ai/sdk

# 5. Generate your API
export ANTHROPIC_API_KEY="sk-ant-..."  # macOS/Linux
# OR
set ANTHROPIC_API_KEY=sk-ant-...       # Windows CMD

node generate-api.js

# 6. Install dependencies
npm install

# 7. Create .env file (see QUICK_START.md)

# 8. Setup database
npx prisma migrate dev --name init
npx prisma db seed

# 9. Start server
npm start

# 10. Visit: http://localhost:3000/api-docs

API ENDPOINTS OVERVIEW
================================================================================

AUTHENTICATION (2 endpoints):
├── POST /api/auth/signup       - Create account
└── POST /api/auth/login        - Login & get JWT

PORTFOLIOS (5 endpoints):
├── POST /api/portfolios        - Create portfolio
├── GET /api/portfolios         - Get all portfolios
├── GET /api/portfolios/:id     - Get one portfolio
├── PUT /api/portfolios/:id     - Update portfolio
└── DELETE /api/portfolios/:id  - Delete portfolio

HOLDINGS (5 endpoints):
├── POST /api/holdings          - Create holding
├── GET /api/holdings?portfolioId=X - Get holdings
├── GET /api/holdings/:id       - Get one holding
├── PUT /api/holdings/:id       - Update holding
└── DELETE /api/holdings/:id    - Delete holding

TRANSACTIONS (5 endpoints):
├── POST /api/transactions      - Create transaction (BUY/SELL)
├── GET /api/transactions?holdingId=X - Get transactions
├── GET /api/transactions/:id   - Get one transaction
├── PUT /api/transactions/:id   - Update transaction
└── DELETE /api/transactions/:id - Delete transaction
